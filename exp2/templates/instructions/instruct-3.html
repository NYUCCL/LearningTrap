Instructions

Each application has four fields, which can have two different values. You can use this information to determine whether an applicant is suitable or not. The four fields and their values are:

Field Value 1 Value 2
Degree: Business Economics
Past employer: Hudson Inc. Nile Co.
Past Position: Developer Analyst
Skill: Design Programming

Each field has a "left" value and a "right" value, but this is only for display purposes. The text is what matters!

Each time you see a new application, the application fields will initially be hidden. Press the SPACE bar to reveal each field one at at time. Once all fields have been revealed, you will be able to click "accept" or "reject".

Please do not use pen and paper or any other external device to help you in this experiment!
